二、这次真正补上的东西

1. 联合动作模型从“结构化乘权”继续推进到“归一化联合动作概率”

`BehavioralLikelihoodModel` 现在不只是保留原来的结构化 fit，
还新增了归一化的联合动作概率分解：

- player probability
- slot probability
- value probability
- continue probability
- joint probability

这一步的意义是：
同一个行为现在会同时被“局部结构是否合理”和“整条动作在候选空间里的相对概率”一起约束。
也就是说，动作似然现在比之前更像一个联合动作分布，而不是纯局部 bonus 乘积。


2. post-hit 搜索从 branch search 推进到带权 expectimax-lite

这次不是只取最强分支了。
现在 rollout 会对 top branches 做带权聚合：

- 用 win probability / continuation likelihood / support ratio / target attack window 做分支权重
- 累积 future expectimax value
- 输出 expected value / margin / support ratio / signal

这让 continue 决策开始读取“整体后续树的期望继续空间”，
而不是只看单条最优分支。


3. benchmark 从固定 case 扩成了更大的扰动样本集

decision benchmark 现在不是 13 个样本，
而是把每个固定 case 扩成 5 个 deterministic variants，
总量提升到 65。

behavior benchmark 也不再只是 5 个静态样本，
而是扩成 base + 噪声 variants，
总量提升到 20。

这还不是大规模 self-play，
但已经不再只是“极少量固定样本”的量化。


4. 抽牌、猜牌、继续停手进一步统一到 strategy objective

这次把 `strategy_objective` 继续往 draw 侧推进了：

- draw rollout 开始累计 `expected_strategy_objective`
- draw color summary 开始直接读取 `strategy_objective_pressure`
- opening plan 现在显式暴露 `strategy_objective`

这意味着 draw / guess / continue 三条线不再只是“共享一些局部信号”，
而是开始围绕同一个总策略目标做比较。
